A modern and responsive News Website built using **HTML, CSS, and JavaScript** that fetches real-time news using **News API**. Users can browse the latest headlines, search for topics, and stay updated with current events through a simple and clean interface.

---

## 🚀 Live Features

✔️ Latest News Updates  
✔️ Search News Articles  
✔️ Category-Based Filtering  
✔️ Responsive Design  
✔️ Dynamic Content Loading  
✔️ Fast and Simple UI  

---

- HTML5

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/your-username/News-Website.git
```

### 2. Open Project

```bash
cd News-Website
```

### 3. Configure API Key

Open your JavaScript file and replace:

```javascript
const API_KEY = "YOUR_NEWS_API_KEY";
```

with your actual News API key.

### 4. Run Application

Open `index.html` in your browser.
